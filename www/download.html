Hosting

We are currently hosted by SourceForge. Both the repository and releases are located there and you should download the releases from the project's site.

You can find there RPMs, debs and an ebuild.

Accessing the repository

If you wish to be most up to date, you can get the sources form the repository: 
$ git clone git://loxim.git.sourceforge.net/gitroot/loxim/loxim
For build instructions, please refer to documentation.
